# Simple Chat App

## TECHNOLOGIES

> ReactJS

> NodeJS / ExpressJS

> LevelDB

## HOW TO RUN THE BACKEND

Go into the `back-end` directory
```
$ npm install
$ node bin/start
```

## HOW TO RUN THE FRONTEND

Go into the `front-end` directory
```bash
$ npm install
$ node bin/start
```

## RESET THE DATABASE

```bash
$ node bin/reset
$ node bin/start
```
